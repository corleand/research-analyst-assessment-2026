# Research Analyst Assessment

This repository contains assessment materials for Research Analyst candidates at M4M.

## Contents

This repository includes:

- **Assessment Instructions** (`Research_Analyst_Assessment_Mar2026.pdf`) - Complete instructions and questions for the assessment
- **Datasets** (3 files in .dta format) - Curated DHS survey data from Ghana, Senegal, and Nepal
- **Metadata Documentation** - Explanatory document describing the data structure
- **Variable Dictionaries** (Excel workbook) - Complete variable definitions for all three datasets
